## Cheapest Possible Stack to Build This

### Hosting — VPS
| Provider | Cost |
|---|---|
| Hetzner Cloud (CX22) | ~$4/month → ~200 EGP/month |
| Contabo VPS S | ~$7/month → ~350 EGP/month |
| DigitalOcean Droplet (basic) | ~$6/month → ~300 EGP/month |

**Hetzner is the best value by far.** 2 vCPU, 4GB RAM, 40GB SSD. More than enough for a clinic site.

---

### Domain
- Namecheap: ~$10/year → ~500 EGP/year

### SSL
- **Let's Encrypt** — completely free, auto-renews every 90 days via Certbot. Zero cost.

---

## Recommended Custom Stack

### Option A — Django + React.js (your strongest stack)

```
Backend:  Django + Django REST Framework
Frontend: React.js (Vite)
Database: PostgreSQL
Server:   Nginx + Gunicorn
SSL:      Let's Encrypt (Certbot)
Hosting:  Hetzner VPS
```

**What you build:**
- Patient appointment booking (calendar UI in React)
- Admin dashboard to manage schedules
- Arabic + English (i18next for React, Django i18n)
- Contact form with email notifications (Django sends via SMTP)
- WhatsApp link integration (just a link, zero cost)
- Basic auth (Django built-in + SimpleJWT)

---

### Option B — Laravel + React.js (also solid)

```
Backend:  Laravel (API mode)
Frontend: React.js (Vite + Inertia or separate SPA)
Database: MySQL
Server:   Nginx + PHP-FPM
SSL:      Let's Encrypt
Hosting:  Hetzner VPS
```

Same features, Laravel gives you Eloquent ORM which is very fast to build booking logic with.

---

No separate frontend framework. Fastest to build. Good enough for a clinic site that doesn't need heavy interactivity.
